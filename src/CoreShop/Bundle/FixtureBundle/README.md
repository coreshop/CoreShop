CoreShop Fixture Bundle
==================

This Bundle is actually a fork of OkvpnMigrationBundle (https://github.com/okvpn/migration-bundle), which is a fork
of OROMigrationBundle. Purpose of this Bundle is to make fixtures easy for CoreShop.

CoreShop
------

![CoreShop](http://www.coreshop.org/assets/img/coreshop-logo.svg)

CoreShop is an eCommerce Solution for Pimcore. It is build from decoupled components to get highest quality of code. [Read more on coreshop.org](http://www.coreshop.org)
